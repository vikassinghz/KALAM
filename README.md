## Overview

Kalam AI is your personal AI writing assistant designed to enhance your writing experience across various domains. From creative content to professional communications, we're here to help you craft the perfect message. Our suite of specialized tools combines the power of artificial intelligence with intuitive design, making it easier than ever to create, edit, and refine your written content.

## Features

*   **Content Creation:** Generate original content on a variety of topics.
*   **Rewriting & Paraphrasing:** Polish existing text to improve clarity and originality.
*   **Email Composition:** Craft professional and effective emails.
*   **Research Assistance:** Generate well-structured research articles.
*   **Social Media Content:** Create engaging posts for platforms like LinkedIn.

## Technologies Used
*   **Frontend:** [React](https://react.dev/), [Vite](https://vitejs.dev/), [TypeScript](https://www.typescriptlang.org/)
*   **AI Integration:** [Groq API](https://groq.com/) for high-speed LLM inference
*   **UI/Styling:** [Tailwind CSS](https://tailwindcss.com/), [Framer Motion](https://www.framer.com/motion/) for animations
*   **Components:** [Lucide React](https://lucide.dev/) for icons
*   **Utilities:** [jsPDF](https://github.com/parallax/jsPDF) & [html2canvas](https://html2canvas.hertzen.com/) for PDF exports

## Setup Instructions

1.  Clone the repository:

    ```bash
    git clone [https://github.com/vikassinghz/KALAM]
    ```
2.  Install dependencies:

    ```bash
    npm install
    ```
3.  Create a `.env` file in the root directory and add your Groq API key:

    ```
    VITE_GROQ_API_KEY=your_groq_api_key_here
    ```
4.  Start the development server:

    ```bash
    npm run dev
    ```

## Usage

1.  Navigate to the application in your web browser.
2.  Select a tool from the navigation bar.
3.  Enter a prompt or topic in the input field.
4.  Click the "Generate" button to generate content.
5.  Review the output and use the export options to save or copy the content.
